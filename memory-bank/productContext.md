# Product Context

## Business Need
- Centralized user management for GC platform
- Secure authentication service
- Scalable user data storage

## User Needs
- Simple registration and login
- Secure account management
- Reliable profile storage

## Key User Flows
1. Registration
   - Create account
   - Set credentials
   - Initial profile setup

2. Authentication
   - Login with credentials
   - Session management
   - Password recovery

3. Profile Management
   - View profile
   - Update information
   - Account settings

## Success Metrics
- Low latency authentication
- High availability
- Secure data storage
- Easy integration with other services
